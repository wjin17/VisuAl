About

VisuAl is a site created to visualize algorithms and as well as a few data structures. This was a fun side project that I wanted to use to learn more about the fundamentals of computer science and web development. The idea was inspired by VisualGo and the projects done by the founder of AlgoExpert. Hopefully you found the examples to be somewhat interesting!
